홈화면 Hero Section을 자동 슬라이드 방식으로 제작해줘.

구성:
- 이미지 2개
- 영상 2개
- 총 4개 슬라이드

슬라이드 순서:
1. 여성 보호자 이미지 + 문구: 엄마 운동 여기가서 해
2. 시니어 4명 이미지 + 문구 없음
3. 상지 움직임 프로그램 영상
4. 낙상예방 움직임 프로그램 영상

동작:
- 페이지 접속 시 자동 재생
- 5초마다 다음 슬라이드로 자동 전환
- 마지막 슬라이드 후 다시 1번으로 반복
- 사용자가 아래 점(dot)을 클릭하면 해당 슬라이드로 이동
- 좌우 화살표는 PC에서만 표시
- 모바일에서는 손가락 스와이프 가능하게

영상 설정:
- 자동재생 autoplay
- 음소거 muted
- 반복 loop
- 배경 영상처럼 표시
- 영상 위에 어두운 반투명 오버레이 적용
- 영상이 로딩되지 않으면 대체 이미지 표시

디자인:
- 전체 화면 가로형 hero
- 높이: desktop 720px, mobile 520px
- 이미지와 영상은 화면 꽉 차게 object-fit: cover
- 글자는 흰색, 크게, 중앙 배치
- 메뉴바는 상단 고정
- BeeMotion 로고와 메뉴가 슬라이드 위에 자연스럽게 보이게

중요:
슬라이드 안에는 불필요한 아이콘, 하단 작은 원형 장식, 임의 텍스트를 추가하지 말 것.
