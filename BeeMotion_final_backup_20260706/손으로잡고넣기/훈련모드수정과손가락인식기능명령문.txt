4. 목적

<제일중요한 손가락인식기능>
최우선 수정 요청: 손가락 인식 기능이 작동하지 않습니다.

현재 문제:
- 카메라 켜기는 됩니다.
- 하지만 손가락 인식 표시가 화면에 나오지 않습니다.
- 초록색 점과 선으로 손 모양이 표시되지 않습니다.
- 손가락을 구부려도 잡기 기능이 작동하지 않습니다.
- 따라서 손으로 잡고 넣기 운동이 구동되지 않습니다.

반드시 먼저 수정할 것:

1. MediaPipe Hands 또는 현재 사용 중인 손 인식 라이브러리가 실제로 로드되는지 확인해 주세요.

2. 카메라 켜기 버튼을 누르면 다음 순서로 작동해야 합니다.

카메라 실행
↓
비디오 화면 표시
↓
손 인식 모델 로드
↓
손이 화면에 들어오면 초록색 점과 선으로 손가락 관절 표시
↓
손 인식 완료 문구 표시
↓
시작하기 버튼 활성화

3. 손 인식 표시

- 손가락 관절 landmark를 초록색 점으로 표시
- 손가락 연결선을 초록색 선으로 표시
- 손이 움직이면 초록 점도 실시간으로 따라 움직이게 처리

4. 손 인식 실패 시 안내문구 표시

“손을 화면에 보여주세요”

손이 인식되면:

“손 인식 완료”

5. 시작하기 버튼 조건

손 인식이 되기 전에는 시작하기 버튼 비활성화

손 인식 완료 후에만 시작 가능

6. 잡기 동작 판정

손가락이 펴진 상태:
- 물체 잡기 안 됨

손가락이 구부러진 상태:
- 잡기 가능

엄지, 검지, 중지 끝점과 손바닥 중심 거리 기준으로
손가락 구부림 여부를 계산해 주세요.

7. 디버그 표시 추가

개발 중에는 화면 한쪽에 아래 상태를 작게 표시해 주세요.

카메라: ON / OFF
손 인식: 성공 / 실패
잡기 동작: 감지 / 미감지
현재 모드: 도움받아 잡기 / 내 힘으로 잡기

8. 가장 먼저 확인할 것

브라우저 콘솔 오류 확인:
- MediaPipe import 오류
- camera_utils 로드 오류
- hands 객체 생성 오류
- video/canvas 연결 오류
- z-index 문제로 canvas가 뒤에 숨겨진 문제
- HTTPS 또는 localhost가 아니어서 카메라 권한이 막힌 문제

9. 완료 기준

카메라를 켰을 때
손을 화면에 보이면
초록색 손가락 점과 선이 반드시 보여야 합니다.

이 기능이 완료되기 전에는
점수, 바구니, 단체모드보다 손 인식 기능을 먼저 수정해 주세요.

필수 조건:
1. 카메라 켜기 후 손 관절 landmark 21개가 초록색 점으로 표시되어야 함
2. 손가락 연결선도 초록색으로 표시되어야 함

잡기 판정 기준:
- 엄지 끝, 검지 끝, 중지 끝이 손바닥 중심에 가까워지면 잡기
- 손가락 끝이 손바닥에서 멀면 손 펼침
- 잡기 상태일 때만 물체가 손에 붙도록 처리
