[열매따기 프로그램 최종 이미지 디자인 수정]

현재 문제:
- 테스트용 나무 이미지 느낌이 강함
- 과일 개수가 적어서 화면이 허전함
- 과일 색상이 흐리고 구분이 어려움
- 실제 과일 느낌이 부족함

수정 목표:
주간보호센터 TV 화면용 고품질 과일나무 이미지로 변경한다.

================================================
배경 수정
================================================

물펌프 관련 이미지는 완전히 삭제한다.
인지훈련 이미지는 나중에 별도로 제작한다.

현재 열매따기 프로그램은:
"예쁜 과일나무에서 열매를 따는 운동" 느낌으로 집중한다.


================================================
나무 디자인
================================================

수정:
- 풍성한 초록색 나무
- 밝은 햇살 느낌
- 입체감 있는 나뭇잎
- 실제 정원 과일나무 분위기

검정 배경 제거.
전체 화면 자연 배경 적용.

스타일:
반실사 3D 일러스트 느낌
어린이 만화 느낌은 줄이고 고급스럽게


================================================
과일 수정
================================================

현재:
딸기, 사과 등 3개 정도만 있음

변경:
화면에 총 6개 과일 배치

사용 과일:

1. 🍎 사과
- 실제 사과 모양
- 빨간색 선명하게
- 광택 효과
- 크기 크게

2. 🍓 딸기
- 진한 빨간색
- 실제 딸기 형태
- 씨앗 표현
- 잎 표현

3. 🍉 수박
- 초록 줄무늬
- 큰 크기
- 둥근 형태

4. 🍌 바나나
- 노란색 선명하게
- 크게 표현

5. 🍇 포도
- 보라색 포도송이
- 알맹이가 보이도록

6. 🥭 망고
- 노랑/주황색
- 실제 망고 느낌


================================================
크기 조정
================================================

큰 과일:
- 사과
- 수박
- 바나나
- 참외

중간:
- 망고
- 포도

작은 과일:
- 딸기


하지만 어르신이 TV에서 보고 잡아야 하므로
모든 과일은 충분히 크게 표시한다.


================================================
배치
================================================

나무 전체에 골고루 배치:

왼쪽 위:
포도

왼쪽 중간:
딸기

가운데:
사과

오른쪽:
수박

오른쪽 위:
망고

아래쪽:
바나나


과일끼리 겹치지 않게 한다.

손 인식으로 잡기 쉽도록
과일 주변 여백 확보.


================================================
게임 기능 유지
================================================

이미지만 변경.
기존 기능 수정 금지.

유지:
- 손 인식
- 잡기 기능
- 점수 계산
- 제한시간
- 개인모드

변경 금지:
JavaScript 동작 코드 수정하지 말 것.


완료 기준:
TV 전체 화면에서 봤을 때
"과일 따기 체험"처럼 보여야 함.

어르신이 멀리서 봐도
사과, 수박, 바나나, 포도 등을 바로 구분할 수 있어야 함.
