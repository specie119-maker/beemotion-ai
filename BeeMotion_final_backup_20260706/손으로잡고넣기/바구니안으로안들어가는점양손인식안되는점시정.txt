손으로 잡고 넣기 운동 - 성공 후 바구니 쌓임 기능 추가

코드가 잘목된 점 분석
exercise.html 분석 결과 수정 요청

1. 양손 인식 수정
현재 maxNumHands: 2로 되어 있지만 실제 코드는 results.multiHandLandmarks[0]만 사용합니다.
results.multiHandLandmarks 전체를 forEach로 돌려서 양손 모두 잡기 판정에 사용해 주세요.

2. 자석기능 강화
도움받아 잡기 모드에서:
magnetRange = 220px
magnetPower = 0.30
으로 수정해 주세요.

현재 150px / 0.15는 너무 약합니다.

3. 자동 성공 방식으로 변경
현재는 물체를 놓은 뒤 떨어질 때만 성공 판정합니다.
수정:
잡은 물체가 바구니 근처 basketMagnetZone에 들어오면 손을 펴지 않아도 자동 성공 처리해 주세요.

4. 바구니 안쪽 쌓임 위치 수정
성공 물건은 basket 전체가 아니라 basketInnerZone 안에만 쌓이게 해 주세요.

basketInnerZone:
x = basket.left + basket.width * 0.28
y = basket.top + basket.height * 0.38
width = basket.width * 0.44
height = basket.height * 0.32

성공 물건 크기는 45~60px로 줄여 주세요.

5. 성공 처리 공통 함수 생성
handleSuccess(obj) 함수를 만들어서:
score += 1
basketItems.push(...)
objects에서 제거
ui-score 업데이트
를 한 번에 처리해 주세요.



현재:
물건을 잡고 바구니 근처로 가져가면 성공 처리할 수 있지만,
성공한 물건이 바구니 안에 쌓여 보이지 않습니다.

수정:

1. 성공 시 물건 삭제 금지

성공한 물건은 화면에서 사라지지 말고
바구니 안쪽에 작게 배치되어 남아야 합니다.

2. basketItems 배열 생성

성공한 물건 정보를 저장해 주세요.

예:
basketItems.push({
  image: object.image,
  x: stackX,
  y: stackY,
  size: 45
});

3. 바구니 안 쌓임 위치

성공할 때마다 바구니 안쪽에 다른 위치로 배치

예:
1개째: 중앙 아래
2개째: 왼쪽 아래
3개째: 오른쪽 아래
4개째: 중앙 중간
5개째: 왼쪽 중간
6개째: 오른쪽 중간

겹치더라도 자연스럽게 쌓인 느낌으로 표시

4. 레이어 구조

바구니 안에 들어간 것처럼 보이게 레이어를 나눠 주세요.

구조:

basketBack
↓
성공한 물건들
↓
basketFront

즉,
물건이 바구니 뒤로 숨거나 바구니 밖으로 보이지 않게
바구니 앞쪽 이미지가 물건 일부를 덮어야 합니다.

5. 바구니 이미지 분리 어려우면 대안

바구니를 분리하기 어렵다면:

성공한 물건을 바구니 입구 안쪽 좌표에
작게 표시하고

z-index를 바구니보다 위에 두되,
바구니 입구 안에 있는 것처럼 보이게 배치해 주세요.

6. 성공 애니메이션

성공 시:

물건이 바구니 안으로 “쏙” 들어가는 애니메이션

처리:
- 물건 위치를 basketCenter로 이동
- 크기를 60%로 줄임
- basketItems에 저장
- 성공 +1

7. 완료 기준

성공할 때마다
바구니 안에 물건이 하나씩 쌓여야 합니다.

성공 1개 → 바구니 안에 1개 보임
성공 2개 → 바구니 안에 2개 보임
성공 3개 → 바구니 안에 3개 보임
