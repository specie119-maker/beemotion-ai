[animal_walk 프로그램 영상형으로 재구성]

목표:
기존 강아지 산책시키기 화면을 재래시장 쇼핑하기처럼
MP4 영상 배경 방식으로 변경한다.

프로그램명:
산책시키기

사용 영상:
assets/animal_walk/walk1.mp4
assets/animal_walk/walk2.mp4
assets/animal_walk/walk3.mp4
assets/animal_walk/walk4.mp4
assets/animal_walk/walk5.mp4
assets/animal_walk/walk6.mp4

중요:
베니스 영상은 사용하지 않는다.

================================================
기본 방식
================================================

재래시장 쇼핑하기에서 성공한 구조를 그대로 사용한다.

걷기 감지됨:
- 현재 산책 영상 play()
- 걸음수 증가

걷기 멈춤:
- 현재 산책 영상 pause()
- 걸음수 증가 중지

영상 종료:
- 다음 walk 영상으로 자동 전환
- walk6 종료 후 다시 walk1로 반복

운동 종료:
- 영상 개수가 아니라 선택한 시간 기준으로 종료

================================================
운동 시간
================================================

선택:
30초
60초
90초

목표 걸음수:
50보
80보
100보

목표 걸음수를 넘어도 종료하지 않는다.
선택한 시간이 끝날 때만 종료한다.

================================================
화면 구성
================================================

전체 화면:
산책 영상

상단 왼쪽:
남은시간

상단 중앙:
산책시키기

상단 오른쪽:
걸음수

오른쪽 하단:
AI 몸 인식창
검정 배경 + 초록색 관절선만 표시

삭제:
- 강아지 PNG 통통 움직임
- 정지 배경 이미지
- 코너 안내창
- 도착 문구
- 불필요한 이벤트 문구

================================================
개발 원칙
================================================

재래시장 쇼핑하기에서 정상 작동한 코드 구조를 복사해서 사용한다.
MediaPipe 몸 인식 코드와 걷기 카운팅 로직은 수정하지 않는다.

변경할 것은:
- 프로그램명
- 영상 배열
- 버튼 시간/목표걸음수
- 결과 문구

================================================
결과 화면
================================================

산책 완료!

표시:
- 총 걸음수
- 운동시간
- 목표 걸음수
- 달성 여부

문구:
목표 달성 시:
“목표를 넘어서 걸었어요!”

목표 미달 시:
“오늘도 잘 걸으셨어요!”
