열매따기 운동을 다시 수정해 주세요.

현재 문제:
1. 손 인식 선이 나무/과일 뒤로 들어가 있음
2. 과일이 12개로 너무 많고 작아서 팔을 위로 올리는 운동이 되지 않음
3. 열매 위치가 낮고 넓게 퍼져 있어 실제 “팔 들어올리기” 효과가 약함

수정 요청:

1. 손 인식 레이어 수정
- 손 랜드마크/손 포인터는 반드시 화면 최상단에 표시
- 그리기 순서:
  배경 → 나무 → 잎 → 과일 → 손 인식 포인터
- z-index:
  handLayer: 9999
  fruitLayer: 50
  treeLayer: 10
- 손 좌표 판정은 이미지 레이어와 상관없이 좌표 거리로 계산

2. 과일 개수 변경
- 화면에 동시에 보이는 과일은 2~3개만 표시
- 과일 크기는 지금보다 크게 표시
- 어르신이 팔을 위로 올려야 닿을 수 있도록 화면 상단에 배치

3. 과일 위치 좌표 수정
- 과일 생성 영역을 화면 상단으로 제한
- 예시:
  y: 화면 높이의 12% ~ 35%
  x: 화면 너비의 20% ~ 80%
- 너무 낮은 위치에는 과일 생성 금지
- 안내문구 박스 근처에는 과일 생성 금지

4. 과일 크기
- 과일 크기: 90px ~ 130px
- 난이도별:
  쉬움: 130px, 판정거리 90px
  보통: 110px, 판정거리 75px
  어려움: 90px, 판정거리 60px

5. 과일 랜덤 종류
과일은 계속 랜덤으로 바뀌게 해 주세요.
사용할 과일 예시:
- 바나나
- 망고
- 딸기
- 복숭아
- 수박
- 참외
- 사과
- 귤
- 포도
- 배

6. 재생성 구조
- 과일을 하나 따면 해당 과일은 아래로 떨어지는 애니메이션 후 사라짐
- 0.5초 후 새 과일이 화면 상단 다른 위치에 생성
- 30초 동안 과일이 계속 생성되도록 함
- 화면에는 항상 2~3개의 큰 과일만 유지

7. 성공 판정
- 손이 과일 가까이에 접근
- 손가락을 오므리거나 주먹 감지
- 아래로 30~40px 당기면 성공
- 성공 시 점수 +1
- 같은 과일 중복 점수 방지

8. 운동 목적 유지
이 프로그램의 목적은 “많이 따기”보다
팔을 위로 올리고 손가락을 오므려 잡는 상지운동입니다.
따라서 과일을 많이 작게 배치하지 말고,
큰 과일 2~3개를 위쪽에 띄워 팔을 들게 해 주세요.
