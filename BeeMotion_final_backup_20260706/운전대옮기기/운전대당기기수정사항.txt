운전대 이동·정지 훈련 버티기 시간 및 목표 이동 수정

현재 문제:
- 한 방향 성공 후 전체 성공 처리됨
- 목표 네모박스가 다음 위치로 이동하지 않음
- 버티기 시간 선택 기능 없음

수정 목표:
오른쪽 → 왼쪽 → 위 → 아래 순서로 진행하고
각 위치마다 정해진 시간 버티기를 완료하면
다음 목표 네모박스로 자동 이동한다.


[버티기 시간 설정]

버튼 추가:
- 10초 (기본 선택)
- 15초

기본값:
holdTime = 10초

사용자가 선택하면:
10초 버튼 → 목표 유지시간 10초
15초 버튼 → 목표 유지시간 15초


[진행 순서]

오른쪽 점선 목표박스 표시

"운전대를 오른쪽으로 이동하세요"

조건:
- 양손으로 운전대 잡기
- 운전대가 오른쪽 목표박스 도착
- 움직임 정지

정지 확인 후:
10초 카운트 시작

화면:
버티는 중
3초 / 10초

"성공! 잘하셨습니다"

↓ 자동 이동

2단계
왼쪽 목표박스 표시

동일 과정 반복

성공 후

↓

3단계
위쪽 목표박스 표시

조건:
양손이 어깨 위 위치까지 올라와야 인정

성공 후

↓

4단계
아래쪽 목표박스 표시

조건:

오른쪽 완료 ✓
왼쪽 완료 ✓
위 완료 ✓
아래 완료 ✓

= 1회 성공

성공점수 +1


[목표박스 이동 규칙]

중요:
목표박스는 동시에 여러 개 보이지 않는다.

현재 목표만 표시:

처음:
오른쪽 □

성공하면:
오른쪽 □ 삭제
왼쪽 □ 생성

성공하면:
왼쪽 □ 삭제
위 □ 생성

성공하면:
위 □ 삭제
아래 □ 생성

성공하면:
1회 완료

steering-wheel.html 수정 요청

현재 문제:
- steering-wheel.html 화면에 개인모드만 보임
- 단체모드 버튼/화면이 보이지 않음
- 사용자가 단체모드로 들어갈 방법이 없음

수정 방향:
steering-wheel.html 안에서 개인모드와 단체모드를 모두 선택할 수 있게 한다.

상단에 모드 선택 버튼 추가:
- 개인모드
- 단체모드

개인모드:
- 기존 기능 유지
- 오른쪽 → 왼쪽 → 위 → 아래 완료 시 1회 성공

단체모드:
- 빨강팀 / 파랑팀 / 초록팀 선택
- 현재 팀 표시
- 현재 참가자 표시
- 다음 사람 버튼
- 팀별 점수 표시
- 결과 보기 버튼

단체모드 점수 규칙:
- 참가자 1명이 오른쪽 → 왼쪽 → 위 → 아래를 모두 완료하면 해당 팀 점수 +1
- 한 방향만 성공해도 점수 올라가면 안 됨
- 4방향 완료해야 1점

버티기 시간:
- 10초 기본
- 10초 / 15초 선택 버튼
- 개인모드와 단체모드 모두 동일 적용

목표박스 이동:
- 오른쪽 성공 후 왼쪽
- 왼쪽 성공 후 위
- 위 성공 후 아래
- 아래 성공 후 1회 성공

추가:
index.html에서만 단체모드가 보이는 구조라면,
steering-wheel.html로 이동했을 때도 단체모드 UI가 반드시 보여야 한다.

권장:
steering-wheel.html 내부에 개인모드/단체모드를 모두 포함한다.
index.html은 프로그램 선택 화면으로만 사용한다.
