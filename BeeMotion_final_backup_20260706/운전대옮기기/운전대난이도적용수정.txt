운전대 이동 훈련 난이도 수정

현재 문제:
- 1단계, 2단계, 3단계가 실제 운동 강도 차이가 없음
- 목표 위치만 동일해서 단계 의미가 없음

수정:
3단계 삭제

난이도는 2개만 사용:
1단계
2단계


================
1단계 (기본 운동)
================

목적:
처음 이용자, 앉아서 운동하는 어르신

운전대 움직임:
- 손 움직임과 운전대 이동을 거의 동일하게 반영
- 가볍게 움직이는 느낌

설정:
movementRatio = 1.0

예:
손 10cm 이동
→ 운전대 10cm 이동

목표 위치:
- 가까운 위치

오른쪽:
몸을 크게 움직이지 않아도 도착

왼쪽:
팔 뻗기 정도

위:
어깨 높이

아래:
허리 아래


================
2단계 (강화 운동)
================

목적:
근력, 보행, 균형 훈련

핵심:
운전대가 무겁고 뻑뻑하게 움직이는 느낌

운전대 이동 감도 감소

설정:
movementRatio = 0.5

예:
손 20cm 이동
→ 운전대 10cm 이동

즉:
더 많이 움직여야
운전대가 목표까지 이동함


목표 위치:
1단계보다 멀게 설정

오른쪽:
- 실제 오른쪽으로 한두 걸음 이동해야 도착

왼쪽:
- 실제 왼쪽으로 한두 걸음 이동해야 도착

위:
- 양손을 어깨보다 확실히 위까지 올려야 도착

아래:
- 다리 방향까지 내려야 도착


추가:
2단계에서는 운전대 이동 속도 제한 적용

예:
손을 빠르게 움직여도
운전대는 천천히 따라옴

maxMoveSpeed 적용


중요:
난이도 차이는 시간이 아님.

잘못된 방식:
1단계 10초
2단계 15초

X

올바른 방식:
1단계 = 가벼운 운전대
2단계 = 무거운 운전대 + 더 먼 목표


화면 수정:
기존:
1단계 / 2단계 / 3단계

변경:
1단계 기본
2단계 강화

버튼 2개만 표시
