폴더 안의 steering-wheel.html 수정 요청

현재 문제:
- steering-wheel.html 화면에 개인모드만 보임
- 단체모드 버튼/화면이 보이지 않음
- 사용자가 단체모드로 들어갈 방법이 없음

수정 방향:
steering-wheel.html 안에서 개인모드와 단체모드를 모두 선택할 수 있게 한다.

상단에 모드 선택 버튼 추가:
- 개인모드
- 단체모드

개인모드:
- 기존 기능 유지
- 오른쪽 → 왼쪽 → 위 → 아래 완료 시 1회 성공

단체모드:
- 빨강팀 / 파랑팀 / 초록팀 선택
- 현재 팀 표시
- 현재 참가자 표시
- 다음 사람 버튼
- 팀별 점수 표시
- 결과 보기 버튼

단체모드 점수 규칙:
- 참가자 1명이 오른쪽 → 왼쪽 → 위 → 아래를 모두 완료하면 해당 팀 점수 +1
- 한 방향만 성공해도 점수 올라가면 안 됨
- 4방향 완료해야 1점

버티기 시간:
- 10초 기본
- 10초 / 15초 선택 버튼
- 개인모드와 단체모드 모두 동일 적용

목표박스 이동:
- 오른쪽 성공 후 왼쪽
- 왼쪽 성공 후 위
- 위 성공 후 아래
- 아래 성공 후 1회 성공

추가:
index.html에서만 단체모드가 보이는 구조라면,
steering-wheel.html로 이동했을 때도 단체모드 UI가 반드시 보여야 한다.

권장:
steering-wheel.html 내부에 개인모드/단체모드를 모두 포함한다.
index.html은 프로그램 선택 화면으로만 사용한다.

추가내용
버티기 성공률 개선 추가 수정

현재 문제:
어르신 손이 미세하게 떨리면
운전대가 목표지점 안에 있어도 정지 상태로 인정되지 않아
타이머가 시작되지 않거나 자주 초기화됨.
이러면 성공률이 낮아지고 성취감이 떨어짐.

수정 방향:
운전대를 잡고 목표지점에 도착하면
미세한 손 떨림은 허용하고
설정된 버티기 시간이 계산되도록 한다.

버티기 판정 변경:
기존:
목표지점 도착 + 거의 완전 정지해야 타이머 시작

변경:
목표지점 도착 + 운전대를 양손으로 잡고 있음
→ 타이머 시작

미세한 흔들림 허용:
- 목표박스 안에서 작은 움직임은 허용
- 손 떨림 때문에 타이머가 초기화되지 않게 한다
- 운전대 중심이 목표박스 안에 있으면 계속 카운트
- 목표박스 밖으로 크게 벗어났을 때만 초기화

권장 설정:
- 목표박스 안쪽 판정 여유값 추가
- 손 좌표 smoothing 적용
- 흔들림 허용 범위 tolerance 적용
- 0.5초 정도 순간 이탈은 봐주기

예시:
10초 버티기 중 6초에서 손이 살짝 흔들림
→ 시간 유지 계속

10초 버티기 중 목표박스 밖으로 크게 벗어남
→ 0초부터 다시 시작

중요:
이 프로그램은 정확도보다 어르신의 성취감이 중요하다.
운전대를 잡고 목표지점에 도착하면
가능한 한 시간이 잘 흐르도록 설계한다.

안내문구:
“좋습니다. 그대로 버텨주세요.”
“조금 흔들려도 괜찮습니다.”
“목표 안에 있으면 시간이 계속 올라갑니다.”


## 버티기 시간 표시 UI 수정

현재 문제:
- 버티기 시간이 가로 막대 형태라 가독성이 낮음
- 어르신이 남은 시간을 직관적으로 보기 어려움

수정 요청:
버티기 시간 표시를 가로 막대에서 시계 모양 원형 타이머로 변경한다.

시계 표시 방식:
- 원형 시계 모양
- 가운데에 큰 숫자 표시
  예: 7 / 10초
- 시간이 흐를수록 원형 테두리가 채워지도록 표시
- 글씨는 크게, 흰색 또는 노란색
- 테두리는 초록색 계열

위치:
- 화면 정가운데는 운전대와 겹칠 수 있으므로 피한다.
- 권장 위치:
  화면 상단 중앙 또는 운전대 바로 위쪽
- 운전대가 목표지점으로 이동해도 시계가 가려지지 않도록 z-index를 높인다.

표시 조건:
- 목표박스에 도착하면 시계 타이머 표시
- 버티기 시작 시 원형 시계가 작동
- 성공하면 “성공!” 표시 후 다음 목표로 이동
- 목표를 벗어나면 시계는 0초로 초기화

문구 예:
시계 안:
7 / 10초

시계 아래:
“그대로 버텨주세요”

중요:
어르신이 한눈에 볼 수 있도록
가로 막대보다 시계형 원형 타이머를 크게 표시한다.


#3steering-wheel.html 단체모드 테스트 가능하도록 수정

현재 문제:
- 현재 steering-wheel.html 파일에서는 개인모드만 테스트 가능함
- 단체모드 버튼 또는 단체모드 화면 전환이 없음
- 팀 점수, 현재 팀, 다음 사람 버튼이 없어 단체모드 검증 불가

수정 요청:
steering-wheel.html 안에서 단체모드까지 바로 테스트 가능하게 만든다.

필수 추가 UI:
1. 개인모드 버튼
2. 단체모드 버튼
3. 빨강팀 / 파랑팀 / 초록팀 선택
4. 현재 팀 표시
5. 현재 참가자 번호 표시
6. 다음 사람 버튼
7. 팀별 방향 완료 기록
8. 팀별 전체 성공 점수
9. 결과 보기 버튼

단체모드 점수 규칙:
- 오른쪽 10초 성공 → 해당 팀 오른쪽 완료 +1
- 왼쪽 10초 성공 → 해당 팀 왼쪽 완료 +1
- 위 10초 성공 → 해당 팀 위 완료 +1
- 아래 10초 성공 → 해당 팀 아래 완료 +1
- 4방향 모두 완료하면 해당 팀 전체 성공 +1점

단체모드 제한시간:
- 10초 선택 시 1인당 50초
- 15초 선택 시 1인당 70초

중요:
개인모드와 단체모드가 같은 운동 로직을 사용하되,
점수 저장 방식만 다르게 한다.

개인모드:
개인 기록에 저장

단체모드:
현재 선택된 팀 기록에 저장

목표:
steering-wheel.html 파일 하나만 열어도
개인모드와 단체모드 둘 다 테스트 가능해야 한다.
