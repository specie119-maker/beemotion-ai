##
손으로 공 넣기 운동 화면에 점수와 시간을 자동 표시해 주세요.

1. 성공 개수 자동 표시
- 물체가 바구니 안에 정확히 들어간 경우에만 성공 개수 +1
- 화면에 항상 크게 표시:
  “성공 0개”
  “성공 1개”
  “성공 2개”

2. 시간 표시
- 사용자가 선택한 시간 10초 / 30초 / 60초가 시작되면 자동 카운트다운
- 숫자로 남은 시간 표시:
  “남은 시간 30초”
- 동시에 시간이 줄어드는 진행바 또는 원형 타이머를 보여 주세요.

3. 위치
- 성공 개수와 시간은 운동 화면을 가리지 않도록 배치
- 추천 위치:
  화면 맨 위 상단바에 넣기

상단바 구성:
왼쪽: 남은 시간 30초
가운데: 손으로 공 넣기
오른쪽: 성공 0개

4. 운동 화면 중앙에는 물체와 바구니만 보이게 해 주세요.
- 시간과 점수는 중앙 운동 영역 안에 크게 겹치지 않게 함
- 여유 공간이나 상단 고정 영역에 표시

5. 시간이 끝나면 자동 종료
- “수고하셨습니다”
- 최종 성공 개수 표시
- 반짝이 효과 출력
- 다시시작하기 버튼 표시
