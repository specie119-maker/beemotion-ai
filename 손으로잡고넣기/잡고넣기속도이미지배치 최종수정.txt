잡고넣기속도이미지배치 최종수정

손으로 잡고 넣기 운동 화면 정돈 및 속도 수정

1. 바구니 위치 수정

현재 문제:
- 바구니가 삐딱하게 보임
- 화면 정중앙 기준이 맞지 않음
- 하단 배치가 정돈되어 보이지 않음

수정:
- 바구니를 화면 하단 정가운데에 배치
- 기울어져 보이지 않게 수평 정렬
- transform rotate 사용 금지
- 좌우 중앙 정렬

CSS 기준:
left: 50%;
transform: translateX(-50%);

- 바구니 아래쪽 여백 일정하게 유지
- 바구니 크기는 현재보다 약간 줄여서 운동 공간 확보


2. 화면 전체 정돈

운동 화면에는 아래 요소만 보이게 정리

- 검정 배경
- 상단 정보바
- 손 인식 초록 점
- 떨어지는 물체
- 하단 중앙 바구니
- 성공 개수
- 남은 시간

안내 박스와 디버그 패널은 운동 시작 후 숨김 처리


3. 물체 내려오는 속도 수정

현재 문제:
보통 속도도 너무 빠름

수정 기준:

보통:
- 1초에 물체 1개씩 생성
- 물체가 천천히 내려오게 설정
- 어르신이 충분히 보고 손을 움직일 수 있는 속도

빠르게:
- 보통보다 약간 빠르게
- 한 번에 여러 개가 정신없이 내려오지 않게 함
- 물체 생성 간격은 0.6초~0.8초 정도
- 내려오는 속도도 보통보다 20~30% 정도만 빠르게


4. 권장 속도값

보통:
spawnInterval = 1000ms
fallSpeed = 1.2 ~ 1.6 px/frame

빠르게:
spawnInterval = 700ms
fallSpeed = 1.8 ~ 2.2 px/frame

주의:
빠르게 모드도 어르신이 잡을 수 있는 수준이어야 함.
게임처럼 너무 빠르게 만들지 말 것.


5. 화면 균형

물체가 내려오는 영역:
화면 상단 15% 아래부터 시작

바구니 영역:
화면 하단 중앙

손 움직임 영역:
화면 중앙 넓게 확보

바구니가 너무 커서 손 운동 영역을 가리지 않게 수정


6. 완료 기준

- 바구니가 화면 하단 정가운데에 수평으로 배치
- 보통 속도는 1초에 1개씩 천천히 내려옴
- 빠르게도 너무 빠르지 않음
- 화면이 어르신 운동 프로그램처럼 정돈되어 보임
