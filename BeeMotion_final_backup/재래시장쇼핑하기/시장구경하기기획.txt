[신규 하지훈련: 시장 구경하기]

목표:
마트에서 카트를 밀고 코너를 구경하며 걷는 느낌의 하지훈련 프로그램을 만든다.
강아지 산책하기의 걷기 인식 엔진을 재사용한다.

================================================
기본 방식
================================================

사용자가 직접 코너를 선택하지 않는다.

프로그램이 랜덤으로 코너를 안내한다.

예:
"생선 코너로 걸으세요"
"과일·야채 코너로 걸으세요"
"우유 코너로 걸으세요"

사용자가 제자리 걷기를 하면:
- 카트가 앞으로 이동
- 걸음수가 증가
- 해당 코너로 이동하는 느낌 표현

일정 걸음수 또는 일정 시간 걷기를 완료하면:
해당 코너 도착 성공 처리

다음 랜덤 코너 안내


================================================
운동 시간
================================================

제한시간:
- 60초
- 90초

30초는 너무 짧으므로 사용하지 않는다.


================================================
코너 목록
================================================

랜덤 코너:

1. 생선 코너
문구:
"생선 코너로 걸으세요"

이미지:
생선, 새우, 오징어, 바다 느낌

2. 과일·야채 코너
문구:
"과일·야채 코너로 걸으세요"

이미지:
사과, 바나나, 포도, 배추, 당근, 브로콜리

3. 우유 코너
문구:
"우유 코너로 걸으세요"

이미지:
우유, 요구르트, 치즈

4. 빵 코너
문구:
"빵 코너로 걸으세요"

이미지:
식빵, 바게트, 쿠키

5. 생활용품 코너
문구:
"생활용품 코너로 걸으세요"

이미지:
휴지, 세제, 샴푸


================================================
화면 구성
================================================

전체 배경:
마트 통로 이미지

하단:
쇼핑카트

상단 왼쪽:
남은시간

상단 오른쪽:
걸음수

중앙 안내창:
랜덤 코너 이미지
+
안내 문구

예:

🐟
생선 코너로 걸으세요


================================================
코너 성공 기준
================================================

한 코너당 성공 조건:

걷기 감지 기준으로
8보~12보 정도 걸으면 성공 처리

처음 버전은 10보 고정 추천

예:
생선 코너로 걸으세요
↓
10보 걷기
↓
생선 코너 도착!
↓
다음 코너 랜덤 안내


================================================
완료 화면
================================================

제한시간 종료 시 결과 표시:

시장 구경 완료!

표시:
- 총 걸음수
- 방문한 코너 수
- 운동 시간

예:
총 82보
4개 코너 구경
운동 시간 60초


================================================
중요 개발 원칙
================================================

복잡한 길찾기 기능 만들지 않는다.
사용자가 실제 방향을 선택하지 않는다.
좌우 이동 기능 넣지 않는다.

걷기만 하면 카트가 앞으로 이동하고,
코너는 랜덤으로 안내된다.

핵심:
시장 구경 느낌 O
복잡한 마트 길찾기 X
