[마트 쇼핑하기 이동감/코너 전환 수정]

현재 문제:
1. 카트가 통통 튀기만 함.
2. 앞으로 가는 느낌이 없음.
3. “과자 코너로 가볼까요?” 문구는 나오지만 배경이 과자 코너로 바뀌지 않음.

수정 목표:
걷기 감지 시 카트가 앞으로 이동하는 느낌을 주고,
목표 코너가 바뀔 때 마트 배경 이미지도 해당 코너 이미지로 변경한다.

구조:
코너별 배경 이미지를 따로 사용한다.

걷기 감지:
walkingState === true 일 때:
- 배경을 아주 천천히 확대(scale 1.0 → 1.08)
- 배경 위치를 살짝 아래쪽으로 이동
- 바닥 라인 또는 진열대가 앞으로 다가오는 느낌
- 카트는 위아래 통통 튀는 효과를 줄이고, 살짝 전진/흔들림만 적용

카트 애니메이션:
기존 통통 튀기 삭제 또는 약하게 변경.
카트는 화면 중앙 하단에 고정.
걷는 중에는:
transform: translateX(-50%) translateY(3px) scale(1.02);
멈춤:
transform: translateX(-50%) translateY(0) scale(1);

코너 성공 기준:
현재 코너에서 10보 걷기 완료 시:
1. “과자 코너 도착!” 표시
2. 방문 코너 수 +1
3. 1초 후 다음 랜덤 코너 선택
4. 배경 이미지를 다음 코너 이미지로 변경
5. 코너별 걸음 카운트 0으로 초기화

중요:
문구만 바꾸지 말고 반드시 배경 이미지도 변경한다.

완료 기준:
- 과자 코너 안내 시 과자 진열대 배경
- 생선 코너 안내 시 생선 코너 배경
- 과일·야채 코너 안내 시 과일·야채 배경
- 걷는 동안 카트가 앞으로 가는 느낌
- 코너 도착 후 다음 코너로 화면 전환
