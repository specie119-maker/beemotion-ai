[로그인/회원가입 오류 수정]

현재 확인:
요금제/결제는 정상 가동됨.
이번 작업에서는 결제 코드 건드리지 말 것.

문제:
1. 로그인 화면에서 Sign Up 클릭 시 회원가입 화면으로 이동하지 않음
2. Login with Google 버튼 클릭이 안 됨

수정 대상:
로그인/회원가입/구글로그인만 수정

수정 금지:
- 요금제 페이지
- PayPal 결제 코드
- 신용카드 결제 코드
- 운동 프로그램 파일

필수 수정:

1. Sign Up 버튼 연결
Sign Up 클릭 시 회원가입 화면 또는 회원가입 모드로 전환되게 한다.

2. Google 로그인 버튼 클릭 가능하게 수정
버튼 위에 투명 레이어가 덮여 있는지 확인.
z-index, pointer-events 문제 확인.

3. Google 로그인 함수 연결 확인
Login with Google 클릭 시
Firebase Google provider 로그인 함수가 실행되어야 한다.

완료 기준:
- Sign Up 클릭 → 회원가입 화면 표시
- Login with Google 클릭 → 구글 로그인 팝업 또는 계정 선택창 표시
- 기존 결제 페이지는 그대로 정상 작동


[회원가입 버튼 추가]

현재 문제:
로그인 화면에 회원가입 버튼 자체가 없음.
사용자가 신규 가입을 할 수 없음.

수정:
로그인 버튼 아래에 회원가입 버튼을 추가한다.

버튼 구성:

[로그인]

[Google로 로그인]

[회원가입]

[비밀번호 찾기]

회원가입 버튼 클릭 시:
- 회원가입 화면으로 이동
또는
- 같은 화면에서 회원가입 모드로 전환

회원가입 화면 필수 입력:
- 이메일
- 비밀번호
- 비밀번호 확인

버튼:
[회원가입 완료]

완료 후:
회원가입 성공 → 로그인 상태로 홈 이동
또는 로그인 화면으로 이동

중요:
기존 결제/요금제 코드는 건드리지 말 것.
이번 작업은 회원가입 버튼과 연결만 수정한다.
