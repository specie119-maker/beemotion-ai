BeeMotion 로그인/회원가입/결제 화면을 수정해줘.

1. 로그인 화면 글씨 크기 확대
- 전체 입력창 placeholder 글씨 22px 이상
- 버튼 글씨 22px 이상
- 제목 “BeeMotion 로그인” 48px 이상
- 모바일에서도 글씨가 작아지지 않게 수정

2. Google 로그인 버튼 수정
현재:
Google로 로그인

변경:
Google 로고 아이콘 + Google로 로그인

- Google 공식 G 로고 아이콘 삽입
- 버튼 왼쪽에 로고, 오른쪽에 텍스트
- 버튼 높이 최소 64px
- 테두리 #E5E5E5
- 배경 흰색

3. 회원가입 버튼 오류 수정
현재 문제:
회원가입 클릭 시 홈화면으로 이동함.

수정:
회원가입 클릭 시 반드시 회원가입 화면 또는 회원가입 모달로 이동하게 해줘.
홈화면으로 이동하지 말 것.

회원가입 화면 필드:
- 이메일
- 비밀번호
- 비밀번호 확인
- 기관명
- 담당자명
- 연락처
- 회원가입 버튼
- Google로 회원가입 버튼

4. 결제 버튼 추가
현재 문제:
월 구독/연간 구독 클릭 후 PayPal 결제와 신용카드 결제 버튼이 제대로 보이지 않음.

수정:
구독 결제 화면에 아래 결제 버튼 2개를 명확하게 표시해줘.

버튼 1:
PayPal로 결제하기

버튼 2:
직불카드 또는 신용카드로 결제하기

5. 결제 화면 디자인
- 첨부한 PayPal 결제 이미지 참고
- 버튼을 크게 표시
- 모바일에서도 한눈에 보이게
- 버튼 높이 최소 60px
- 글씨 크기 22px 이상
- PayPal 버튼은 파란색 계열
- 카드 결제 버튼은 진한 회색 또는 흰색 테두리 버튼

6. 결제 연결 규칙
- 무료체험 시작: 결제페이지로 이동 금지, 훈련 프로그램 화면으로 이동
- 월 구독하기: 월 구독 결제 화면으로 이동
- 연간 구독하기: 연간 구독 결제 화면으로 이동

- 지금은 테스트 환경이면 “Sandbox 테스트 결제”라고 표시
- 실제 결제에서는 PayPal Checkout 버튼과 Card Checkout 버튼이 모두 보여야 함
