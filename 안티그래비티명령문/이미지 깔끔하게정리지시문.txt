이미지 깔끔하게 정돈 지시문

-손으로 잡고 넣기 운동 이미지 수정

현재 떨어지는 이미지와 바구니 이미지 배경 제거 필요

문제:
- 이미지 주변 흰색 사각 배경 표시됨
- 이미지 (예 ; 별모양) 밑에 하얀줄 배경표시됨
- 아래쪽 흰색 그림자 영역 표시됨
- 카메라 화면과 자연스럽게 합성되지 않음


수정:

1. 모든 운동 오브젝트 이미지 누끼 처리

대상:

- 공
- 별
- 강아지
- 판다
- 한국돈
- 달러
- 바구니

모두 적용


2. 이미지 형식 변경

JPG 사용 금지

반드시:

PNG
transparent background

사용


3. CSS 처리 확인

이미지 태그 배경 제거:

background: transparent;

적용


4. 흰색 제거

RGB 기준:

#FFFFFF
#FAFAFA
밝은 회색 배경 제거


5. 그림자 수정

기존 이미지에 포함된 흰색 그림자 삭제

필요하면 CSS shadow로 다시 적용

예:

filter:
drop-shadow(0 8px 15px rgba(0,0,0,0.25));


6. 화면 표시 기준

정상:

카메라 영상 위에 물체만 보임

잘못:

하얀 네모 박스
흰색 받침
흰색 테두리


7. 바구니 수정

현재 바구니 아래 흰색 영역 삭제

바구니만 남기기

투명 PNG 처리


8. 성공 후 쌓이는 이미지도 동일 적용

바구니 안에 들어간 작은 이미지도
투명 PNG 유지


완료 기준:

카메라 화면 위에
실제 물체가 떠 있는 것처럼 보여야 함
