BeeMotion 홈 Hero 영상 적용 방식을 수정해줘.

BeeMotion 홈화면 Hero 슬라이드/영상 중 4번째 영상이 잘못 들어가 있습니다.

수정 요청:
- 현재 홈화면 4번째 영상을 삭제
- 새로 첨부한 영상 파일로 교체

교체할 영상 파일:
엄마운동여기가서해.mp4

중요:
이번에 첨부한 영상 “엄마운동여기가서해.mp4”는
영상 자체에 문구가 들어 있으므로,
이 영상 위에는 BeeMotion 제목, 부제, 버튼을 절대 겹쳐 올리지 말 것.

수정 요청:
1. 홈화면 Hero의 4번째 슬라이드/영상은
   “엄마운동여기가서해.mp4” 영상만 단독으로 보여줘.

2. 4번째 영상 위에는 아래 요소를 모두 숨김 처리:
   - BeeMotion 큰 제목
   - 시니어 AI 움직임 프로그램
   - 즐겁게 움직이고 건강하게 생활하세요
   - 무료체험 시작 버튼
   - 데모 시청 버튼

3. 4번째 영상에서는 영상 자체만 전체 화면으로 재생되게 해줘.
   - object-fit: cover
   - width: 100%
   - height: 100vh 또는 Hero 높이에 맞춤
   - autoplay
   - muted
   - loop
   - playsinline

4. 1~3번째 Hero 영상/이미지는 기존처럼
   BeeMotion 제목, 부제, 버튼이 보이도록 유지해줘.

5. 4번째 영상에만 예외 처리:
   if currentSlide === 4:
      overlay text/buttons hidden
   else:
      overlay text/buttons visible

중요:
4번째 영상은 사이트 문구와 겹치지 않게,
영상 단독으로만 사용해야 합니다.
