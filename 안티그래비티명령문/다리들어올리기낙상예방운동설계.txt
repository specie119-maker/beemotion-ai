튼튼다리 운동 판정 기준을 다시 수정해 주세요.

현재 무릎을 위로 올리는 방식은 어르신에게 무리가 됩니다.
의자에 앉은 상태에서 무릎을 높이 드는 것이 아니라,
다리를 앞으로 펴고 발을 살짝 들어올리는 방식으로 변경해 주세요.

수정 범위:
- 튼튼다리 운동만 수정
- 다른 프로그램 수정 금지
- 메뉴/라우팅/팀모드/타이머 구조 수정 금지

수정 내용:

1. 운동 방식 변경

기존:
앉은 상태에서 무릎을 들어올림

변경:
앉은 상태에서 다리를 앞으로 펴고,
발끝/발목을 살짝 들어올리면 성공

오른발:
“오른쪽 다리를 앞으로 펴고 발을 살짝 들어 올려주세요”

왼발:
“왼쪽 다리를 앞으로 펴고 발을 살짝 들어 올려주세요”

두발:
“양쪽 다리를 앞으로 펴고 발을 살짝 들어 올려주세요”

2. 판정 기준 변경

무릎 Y좌표 상승을 주 기준으로 하지 말 것.

대신:
- 발목/발끝 Y좌표 상승
- 무릎은 크게 움직이지 않아도 됨
- 다리가 앞으로 펴진 상태에서 발목이 기준보다 위로 올라가면 성공

오른발 성공:
오른쪽 발목 또는 발끝이 기준선보다 위로 올라갔다가 내려오면 1회

왼발 성공:
왼쪽 발목 또는 발끝이 기준선보다 위로 올라갔다가 내려오면 1회

두발 성공:
양쪽 발목 또는 발끝이 함께 올라갔다가 내려오면 1회

3. 기준선 완화

어르신이 힘들지 않도록 높이 기준을 낮게 설정

권장:
- 발목 Y좌표가 기준 위치보다 20~35px 정도만 위로 올라가도 성공
- 완전히 높게 들 필요 없음
- 1회당 3초 초과 시 다음 시도로 넘어가기

4. 화면 디자인 수정

중앙 의자 그림이 너무 높아 보이면 낮은 의자로 변경하거나,
의자 이미지를 단순화해 주세요.

화면 안내:
“높이 들지 않아도 됩니다. 다리를 앞으로 펴고 발을 살짝 들어 주세요.”

5. 성공 안내

성공 시:
“좋아요!”
“무리하지 않고 잘하셨어요!”

중요:
이 운동은 무릎을 높이 드는 운동이 아닙니다.
앉아서 다리를 앞으로 펴고 발을 살짝 들어올리는 안전한 하지운동입니다.


## 튼튼다리 운동의 신체인식 표시 방식을 부드럽게 수정해 주세요.

수정 범위:
- 튼튼다리 운동의 Pose 표시 UI만 수정
- 실제 인식/카운트 로직은 유지
- 다른 프로그램 수정 금지

현재 문제:
- 신체인식 선이 너무 길게 표시됨
- 화면이 복잡해 보임
- 다리 움직임보다 긴 초록 선이 더 눈에 띔

수정 요청:

1. 전체 뼈대선 표시 줄이기
- 전신 연결선 전체를 모두 그리지 말 것
- 팔/몸통/머리 연결선은 숨김 처리
- 다리 운동에 필요한 관절만 표시

표시할 관절:
- 왼쪽 무릎
- 오른쪽 무릎
- 왼쪽 발목
- 오른쪽 발목
- 필요 시 골반 좌우 점

2. 선 표시 방식 변경
- 긴 선 대신 짧은 선으로 표시
- 무릎 ↔ 발목 구간만 표시
- 선 두께는 얇게
- 불투명도 낮게

3. 관절 점 표시
- 관절은 작은 점으로 표시
- 손가락 인식처럼 부드럽게 움직이도록 smoothing 적용
- 점 크기 8~12px 정도

4. 움직임 표시
- 발목이 올라갈 때 짧은 위쪽 화살표 또는 부드러운 점 효과 표시
- 성공 시 발목 주변에 원형 효과 표시

5. 좌표 smoothing 적용
- 이전 좌표와 현재 좌표를 바로 바꾸지 말고 보간 처리

예:
smoothX = previousX * 0.7 + currentX * 0.3
smoothY = previousY * 0.7 + currentY * 0.3

6. 화면 목적
- 사용자가 “인식되고 있구나”만 알 수 있으면 됨
- 긴 뼈대선으로 전신을 강조하지 말 것
- 다리 움직임만 부드럽고 간단하게 보여줄 것

중요:
신체인식 판정 로직은 유지하고,
화면에 보이는 Pose 시각화만 간단하고 부드럽게 수정해 주세요.
