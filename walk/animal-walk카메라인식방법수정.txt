[동물 산책시키기 카메라 화면 표시 방식 수정]

현재 문제:
카메라 미리보기 화면에 실제 사람 얼굴과 주변 환경이 그대로 표시됨.

수정 목표:
카메라 영상은 내부적으로만 사용하고,
사용자에게 보이는 화면은 AI 몸 인식 화면으로 변경한다.

================================================
카메라 표시 방식
================================================

기존:
웹캠 원본 영상 표시
(얼굴, 방, 사람 모습 그대로 노출)

삭제

변경:

검정 배경 화면 위에
MediaPipe Pose Skeleton만 표시한다.

표시 요소:
- 관절 포인트
- 연결선

디자인:
배경:
검정색

관절 점:
초록색 원

연결선:
초록색 선

표시할 부위:
- 어깨
- 팔
- 몸통
- 골반
- 다리

얼굴 세부 표시 제거:
눈, 코, 입 점 표시하지 말 것.

얼굴 landmark:
숨김 처리

사용:
Pose landmark만 시각화

예:

      ●
      |
  ●---●---●
  |       |
  ●       ●
      |
     ● ●
     | |
     ● ●


================================================
인식 상태 표시
================================================

정상:
초록색 표시

문구:
"AI 몸 인식 완료"

인식 안됨:
"조금 뒤로 이동해주세요"


================================================
중요
================================================

운동 분석에는 원본 카메라 사용 가능.

하지만 화면 출력:
video element 표시 금지

canvas 사용:
검정 배경
+
pose skeleton draw

사용자가 보는 화면에는
실제 얼굴/집안 배경/사람 영상이 나오지 않도록 한다.


목적:
- 개인정보 보호
- TV 화면 집중도 향상
- 전문 운동기기 느낌 제공
